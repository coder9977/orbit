# Backend service

Koa2 + Node.js + MongoDB backend with Apollo GraphQL.

## Run locally

```bash
cd Backend
bun install
bun run dev
```

## Environment

- `PORT` — backend port, default `4000`
- `MONGO_URI` — MongoDB URI, default `mongodb://127.0.0.1:27017/koa_backend`

## GraphQL endpoint

- `http://localhost:4000/graphql`
